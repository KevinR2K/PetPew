 PetPew 

A full-stack pet shop e-commerce website where users can browse 
and purchase pets and pet accessories online.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

 Features (in progress)
- [x] Homepage with product categories
- [ ] Product pages (Dogs, Cats, Birds, Aquarium, Exotic)
- [ ] User authentication
- [ ] Shopping cart & checkout
- [ ] Order tracking
- [ ] Admin dashboard

## Database
Import `petpew.sql` into your MySQL server to set up the database.

 Setup
1. Clone the repo
2. Import `petpew.sql` into MySQL
3. Update `api/config.php` with your database credentials
4. Run on a local server (XAMPP / WAMP)
